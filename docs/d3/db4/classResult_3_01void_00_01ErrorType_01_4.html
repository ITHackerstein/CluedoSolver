Detailed Description

template<typename ErrorType>
class Result< void, ErrorType >

A specialization of the Result class for the case where the value is void.

This class is basically an alias for std::optional<ErrorType> with the exception it can be used with the TRY and MUST macros.

Template Parameters
ErrorTypeThe type of the error that the Result object may contain.
